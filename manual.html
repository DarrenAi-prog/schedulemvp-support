Generate a Smart Week (Smart Draft)

  1. Go to Schedule
  2. Tap Smart Week
  3. Select your Location and confirm the week range
  4. Pick a shift template (start/end) and choose which weekdays to include
  5. Under each role, set Needed (how many people you need per day)
  6. Tap Smart Draft
  7. Review the draft, then tap Create to add the shifts to your schedule

Tips

  • If a role has no eligible employees, Smart Draft will leave open slots.
  • Smart Draft won’t overwrite days that already have shifts.
  • Smart scheduling avoids double-booking employees in the same time window.

How swaps work

Swaps are a 2-step approval to prevent confusion and keep managers in control.

  1. Employee A requests a swap
    • Picks their shift to trade away
    • Picks a coworker
    • Picks a specific coworker shift they want in return
  2. Coworker accepts/declines
    • If accepted, it goes to the manager for final approval
  3. Manager approves/denies
    • If approved, ScheduleMVP swaps who is assigned to the two existing shifts

Swap rules

  • Swaps must be same location.
  • Swap choices only show shifts that don’t overlap the requester’s existing schedule.

Time-off requests

Employees: My Schedule → Request time off

Managers: Requests → Time Off → Approve/Deny

Push notifications

In the app: Settings → Notifications → Preferences

  • Time-off requests
  • Time-off approved/denied
  • New messages
  • Schedule changes

Note: iOS also has a global switch in iPhone Settings → Notifications → ScheduleMVP.
